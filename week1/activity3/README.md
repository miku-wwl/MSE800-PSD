# Activity 3: Power Calculator

## Overview

This is the exercise project in `week1/activity3`, containing a simple Python program `power_calculator.py`. The program calculates the value of an integer `x` raised to the power of `y`.

## Files

- `power_calculator.py`: Main program file. It prompts the user to enter two integers, `x` and `y`, then calculates and prints `x^y`.
- `screenshot.png`: A screenshot of the activity or program output (for reference).

## Usage

1. Open a terminal and change to the directory:
   ```bash
   cd week1/activity3
   ```
2. Run the program:
   ```bash
   python power_calculator.py
   ```
3. Enter the values when prompted:
   - `x`: base number
   - `y`: exponent

The program will display the calculated result of `x^y`.

## Example

If you enter:

- `x = 2`
- `y = 5`

The output will be:

```text
2^5 = 32
```

## Learning Goals

This exercise focuses on:

- Python input and output
- Converting input strings to integers
- Exponentiation using `**`
- Defining and calling a function
