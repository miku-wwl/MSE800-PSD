# Week 3 Activities

This folder contains Week 3 activity submissions.

Conventions used in the ER diagrams:

- Attributes labeled `PK` are primary keys.
- Attributes labeled `FK` are foreign keys.
- Relationship cardinalities are shown as `1:1`, `1:N`, or `M:N` (via a bridge table).
