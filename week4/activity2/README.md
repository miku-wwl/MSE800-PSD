# Week 4 - Activity 2: OOP Project (Area and Perimeter)

## Task

Develop an object-oriented programming (OOP) project that allows users to input the dimensions of a **rectangular piece of land** and calculate its **area** and **perimeter**.

Requirements satisfied:

- At least **two** `.py` files
- Includes **one class**
- Implements **a couple of methods** (area + perimeter)

## Project Structure

- `src/rectangle_land.py` - defines the `RectangleLand` class
- `src/main.py` - user input + prints results
